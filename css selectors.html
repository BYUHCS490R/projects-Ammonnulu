List out the different ways to select an item.

Tag name, ID, class, children

What is the difference between an ID and a Class?

ID is for one element only. Class is for many elements.

What is the difference between a SPAN tag and a DIV tag?

SPAN is Inline, for small text parts inside a line. DIV is Block, for big sections.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV is just a box, no meaning. HEADER/MAIN/FOOTER is Have meaning top, middle, bottom parts of page.

What are the different ways to select a color?

Name, Hex codes, RGB, RGBA, HSL, HSLA
