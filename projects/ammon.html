Internet

Internet is big network that spread signals that connects computers, phones and other devices that can able to access internet.

Web

The Web is made of websites and pages that we open in a browser like Chrome.

How are the Internet and Web different?

The internet is a wires and network. The Web is one service on it that shows sites and pages.

ASCII and Unicode

ASCII is an old system to show letters and numbers with codes. Unicode is newer and can show characters from all languages.

What is UTF-8

UTF-8 is a way to save and send Unicode text. It works for almost all languages.

URL

A URL is the web address you type, to open a site.

Domain Name

A domain name is the simple name of a site, like yahoo.com, instead of numbers.

HTTP

HTTP is the rule that sends web pages from a server to your browser.

HTTPS

HTTPS is the safe version of HTTP. It hides your data with security.

IP Adress

IP address is a number that shows where a computer or phone is on the Internet.

DNS

DNS is like a phone book. It changes a domain name into the right IP address.

W3C Recommendations

W3C Recommendations are rules made by a group that guide how websites should be built.

XML, XHTML, HTML, HTML5

XML is used to store data. XHTML is a stricter form of HTML. HTML is the main code for web pages. HTML5 is the newest version with more tools for modern sites.
