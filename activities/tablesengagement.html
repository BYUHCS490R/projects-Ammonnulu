Why don't we use the border attribute?

We do not use the border attribute because it is an old HTML method. It is better to use CSS for styling the table, as it gives more control and looks cleaner.

What is the easiest way to color every other row a different color?

We can use the :nth-child() selector in CSS. It will automatically color every second row.
