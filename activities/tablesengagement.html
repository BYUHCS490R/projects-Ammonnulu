When do you use a table?

Use a table when you want to show information or numbers in a clear way using rows and columns, like marks, prices, or timetables.

When do you not use a table?

Do not use a table if you are only listing few points or using it just to arrange text or pictures.

When do you need to do extra work for table accessibility?

Do extra work when your table has many headers or mixed cells, so that screen readers can read it properly for all users.
