Explain the difference between inline, embedded, and external CSS.

Inline CSS is style written inside the HTML tag. Embedded CSS is style written inside style in the HTML file. External CSS is style written in a separate file and linked.

Explain why you should only use external CSS.

Easy to reuse on many pages. Easy to update in one place. Keeps code clean.Makes website faster.

What is a selector used for?

A selector tells CSS which HTML element to style.
