What is normal flow in HTML?

Normal flow means the default order in which things appear on a webpage.

What does float do? Why do you need to clear the float?

The float property is used to move an element to the left or right side of the page. We use clear float because sometimes floating makes the next elements go out of place.

What do absolute, fixed, and relative do?

Relative moves the element a little from its normal place. Absolut puts the element in an exact position on the page or inside another box. Fixed keeps the element in the same place even when you scroll the page.

How can you make nice navigation buttons for mobile devices?

Make buttons big enough to touch easily. Use padding and rounded corners. Keep colors simple, like blue or green with white text. Arrange buttons using flexbox or grid so they look neat. Add small icons for a better look.
